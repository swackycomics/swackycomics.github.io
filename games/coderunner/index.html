CODERUNNER.exe

Date released: July 13, 2021

Last update: March 17, 2026

Platforms: Windows

Game engine: Clickteam Fusion 2.5

Status: Complete

Welcome to CODERUNNER. Play as "Not THE .EXE GUIDER", and collect all the scattered code pieces across eleven stages of FUN! Use arrow keys to walk around and Z to dig. Sometimes code can be hidden in the ground!

Initially featured as its own game in THE .EXECUTIONER. I've decided to make a standalone version so that this whole thing is easier to access. This new version shows Boss' HP.
